
# Sentiment Analysis: VADER vs RoBERTa

Comparing two sentiment analysis approaches on Amazon product reviews.

## Overview
- Used VADER (rule-based) and RoBERTa (transformer-based) to score 500 Amazon reviews
- Visualised how each model's scores compared against actual star ratings
- Explored edge cases where the two models disagreed

## Tools
Python · pandas · NLTK · HuggingFace Transformers · seaborn · matplotlib
